# Paginex Android Project

This repository contains the Android Studio project for Paginex.

## Requirements

- Android Studio (latest stable recommended)
- Android SDK Platform 34
- JDK 17 (usually bundled with Android Studio)

## Run

1. Clone the repository.
2. Open the project root in Android Studio.
3. Let Gradle sync complete.
4. Run the `app` configuration on an emulator or physical device.

`local.properties` is intentionally not tracked. Android Studio will create it automatically on first open.
